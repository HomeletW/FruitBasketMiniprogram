# 管理员使用指南

## 登录

1. 进入 `/pages/admin-login/index`（隐藏入口）。
2. 输入管理员账号与密码。
3. 登录成功后会将管理员 token 存入本地存储，用于后续接口调用。

## 订单管理

- 在管理面板的 **订单管理** 区块更新订单状态。
- 合法的状态流转请参考 `docs/SCHEMA.md` 中的状态机说明。

## 查看订单详情

- 管理面板仅展示订单列表，细节查看可在 CloudBase 控制台调用 `adminGetOrderDetail`。

## 安全提示

- 管理员账号信息仅存放于 `admins` 集合。
- 需要更新 token 时重新登录即可。
